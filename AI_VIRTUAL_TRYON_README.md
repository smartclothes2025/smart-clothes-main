# 🎨 AI 虛擬試衣功能說明

## ✨ 功能亮點

### 從簡單圖形到逼真試穿
- **之前**: 使用圓圈、線條和矩形的簡單 SVG 圖形
- **現在**: AI 生成的專業時尚攝影級別試穿圖片

### 兩種試穿模式

#### 1. 標準模式 🤖
- 使用專業時尚模特兒
- 根據您的身體數據調整體型
- 高質量時尚攝影風格
- 適合快速預覽穿搭效果

#### 2. 個性化模式 📸
- 上傳您自己的照片
- AI 分析您的外貌特徵
- 生成更貼近您本人的試穿效果
- 更真實的個人化體驗

---

## 🚀 使用步驟

### 步驟 1: 選擇服裝
1. 進入「衣櫃」頁面
2. 選擇想要試穿的服裝單品
3. 點擊「虛擬試衣」按鈕

### 步驟 2: 輸入身體數據（可選）
- 身高、體重
- 胸圍、腰圍、臀圍
- 肩寬
- 這些數據幫助 AI 生成更符合您體型的圖片

### 步驟 3: 選擇試穿模式

#### 標準模式（推薦新手）
- 直接等待 AI 自動生成
- 10-30 秒即可看到結果

#### 個性化模式（更真實）
1. 勾選「使用我的照片」
2. 上傳一張清晰的全身或半身照
3. 點擊「重新生成」
4. 等待 AI 分析並生成個性化試穿圖

### 步驟 4: 查看和保存
- 查看 AI 生成的試穿效果
- 如果不滿意，可以點擊「重新生成」
- 填寫標題和描述
- 選擇是否同步到貼文
- 點擊「保存穿搭」

---

## 💡 使用技巧

### 獲得最佳效果的建議

#### 照片要求（個性化模式）
✅ **推薦**:
- 清晰的全身或半身照
- 光線充足
- 背景簡潔
- 正面或側面角度
- 穿著貼身或簡單的服裝

❌ **避免**:
- 模糊或低解析度照片
- 過暗或過亮
- 複雜背景
- 遮擋面部或身體
- 已經穿著複雜服裝

#### 服裝選擇建議
- 選擇風格協調的單品
- 注意顏色搭配
- 考慮季節和場合
- 不要選擇過多單品（3-5 件最佳）

#### 身體數據建議
- 盡量填寫準確的數據
- 至少填寫身高和體重
- 數據越完整，效果越好

## 🔧 故障排除

### 問題 1: 顯示「AI 生成服務未配置」
**原因**: 後端未配置 AI API

**解決**: 
- 聯繫管理員配置 API Keys
- 參考 `backend/AI_SETUP_GUIDE.md`

### 問題 2: 生成時間過長
**原因**: AI 生成需要時間

**解決**: 
- 耐心等待 10-30 秒
- 檢查網路連接
- 如果超過 1 分鐘，嘗試重新生成

### 問題 3: 生成的圖片不理想
**解決**: 
- 點擊「重新生成」嘗試不同效果
- 調整身體數據
- 更換服裝組合
- 使用個性化模式上傳照片

### 問題 4: 照片上傳失敗
**解決**: 
- 確認照片格式（JPG, PNG）
- 確認照片大小（建議 < 5MB）
- 嘗試壓縮照片
- 更換其他照片

## 🔐 隱私保護

### 我們的承諾
1. **照片不儲存**: 上傳的照片僅用於生成，處理完立即刪除
2. **數據加密**: 所有傳輸使用 HTTPS 加密
3. **本地處理**: 身體數據僅儲存在您的帳戶中
4. **可選功能**: 照片上傳完全可選，您可以選擇標準模式

### 數據使用說明
- 上傳的照片: 僅用於 AI 分析，不會儲存或分享
- 身體數據: 儲存在您的帳戶中，僅您可見
- 生成的圖片: 可選擇是否分享到貼文
